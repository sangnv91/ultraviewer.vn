Extra Description


Number of users and devices licensed

Each license can be used by one user at a time. With the Professional plan, each user can use UltraViewer on up to six devices. If you need to use the software for multiple users simultaneously or install it on more devices, you can purchase additional licenses.

Remote Endpoints

Each computer has its own ID number, when you connect to a new ID number that within 1 month (~30 days) you have never controlled, UltraViewer will count as a new device.
