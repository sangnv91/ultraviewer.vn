arko - 10/22/2018 1:14:55 AM

Brandon,
You can do that with command line:
Create a bat file COMPUTER1.BAT with command line in it "UltraViewer_Desktop.exe -i:ID_TO_CONNECT -p:PASSWORD"
Create second bat file COMPUTER2.BAT with command line in it "UltraViewer_Desktop.exe -i:ID_TO_CONNECT -p:PASSWORD"
And so on...
You will have several bat files, and when you click it will automatically connect to them.
