How to send files on UltraViewer?

  • Tracy Tran
  • 35 comments

Question:

How to send files via UltraViewer?
 

Answer:

While you are connected to the remote computer, you can utilize the chat feature within UltraViewer to transfer files. To send a file, please follow this instruction:
- While remote controlling your partner computer, press F1 to toggle the chat box on.
- Drag the file you want to send to the chat box and send a message.
- On your partner computer, click on the Save button to save the file.

We are currently working on creating a more convenient way to send files via the copy and paste clipboard. This feature will be available in the next version soon. 
If you have multiple files or folders. Please compress the file using zip/rar and send the zipped file. We will support multiple file copies soon.
